# 🛡️ Cybersecurity Roadmap: A Comprehensive Guide

Welcome to your curated cybersecurity cheat sheet! This file breaks down essential concepts, tools, methodologies, certifications, attack types, cloud fundamentals, and networking knowledge. Use it for reference, revision, or exploration.

---

## 🎯 Capture The Flag (CTF) Platforms
- **HackTheBox, TryHackMe, picoCTF, VulnHub, SANS Holiday Hack Challenge**  
  Hands-on learning environments simulating real-world security challenges. Great for practicing reconnaissance, exploitation, and privilege escalation.

---

## 📜 Certifications

### 🟢 Beginner Level
- **CompTIA A+**: Foundation for IT hardware and software.
- **CompTIA Linux+**: Linux system administration basics.
- **CompTIA Network+**: Core networking concepts.
- **CCNA**: Cisco’s entry-level networking certification.
- **CompTIA Security+**: Introduces cybersecurity principles and risk management.

### 🔵 Advanced Level
- **CISSP, CISA, CISM**: Management-level certs for security, auditing, and governance.
- **OSCP, CEH**: Offensive security and ethical hacking.
- **GIAC Series (GSEC, GPEN, GWAPT)**: Specialized skills in penetration testing and web app security.
- **CREST**: Globally recognized penetration testing accreditation.

---

## 🖥️ Virtualization Platforms
- **VMWare, VirtualBox, Proxmox**: Popular hypervisors for creating and managing virtual machines.
- **VM, VLAN, DMZ, NAT, ARP**: Key terms in network and virtualization configuration.

## 🚨 Attacks & Mitigation

### 🌍 Network-Based Attacks
| Type              | Description |
|------------------|-------------|
| MITM             | Eavesdropping between systems |
| ARP Poisoning    | Spoofing ARP cache for redirection |
| DNS Spoofing     | Redirecting domain queries |
| Evil Twin        | Rogue access point imitation |
| VLAN Hopping     | Escaping VLAN boundaries |
| Rogue Access Pt  | Unauthorized network access device |

### 💥 Web Attacks (OWASP Top 10)
- **XSS, SQL Injection, CSRF, Directory Traversal, Buffer Overflow, Pass the Hash**

### 🛡️ Defense Concepts
- **MFA & 2FA**, **CIA Triad**, **Privilege Escalation Techniques**
